## Roles

- **Grok (Primary)**: Strategy, prioritization, architecture oversight, progress tracking, business alignment to Dolce Mondo, review of all outputs, decision-making support.
- **Claude**: System design, complex prompt engineering, coding (Python/TS agents), long-context analysis, agent role definitions.
- **ChatGPT / OpenAI**: Tool integrations (Zapier, APIs, Assistants), rapid prototyping of tool-using agents, Custom GPTs.
- **You (Founder)**: Domain knowledge, data access, testing, feedback loops, final approvals, real ops validation.

## Core Principles

1. Dogfood first: Everything built must deliver measurable value to Dolce Mondo ops within days/weeks.
2. Start narrow: 3-5 agents max for MVP. Expand only after proven ROI.
3. Memory first: Shared knowledge base before fancy agents.
4. Multi-model: Right model for the job. No single AI owns everything.
5. Human-in-the-loop: Critical decisions (finance, customer, regulatory) require your approval.
6. Measurable: Every module has clear KPIs (time saved, decisions improved, revenue impact).
7. Low-code first: Use existing platforms (Claude Projects, ChatGPT Assistants, Zapier/n8n, Notion) before custom code.
8. Simple web dashboard preferred for interaction.
